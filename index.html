About me

Helloooh!
My name is Daniel and I enjoy building web applications. My interests in web development started back in 2019 , when I was a senior at San Francisco State University. I was a pretty broke college student, and I wasn't able to afford tutoring when my upper division Japanese course work started getting difficult. I knew I needed to do something to get help for free, and little did I know that desperate situation would lead me to creating a pretty humble language learning platform that is being used by students worldwide. Curious? You can visit and learn more by going to the project section down below!

I thought myself the basics of web development, but I realized there were some gaps in my knowledge. In an effort to fill in those gaps, I decided to invest in myself and my professional career by signing up to a coding bootcamp. There I'll be able to solidfy my foundations, and learn new technologies needed to be a competitive jr. software engineer.

Here are a few technologies I've been working with recently:

  • JavaScript (ES6+)
  • React
  • Node.js
